# 🌦️ Weather App

A modern and feature-rich weather application. Users can search for cities, use their current location, view hourly and 5-day forecasts, save favorite cities, and check additional data like air quality.

---

## ✨ Features

- 🔍 Search weather by city
- 📍 Get weather by current location
- ⏱️ Hourly forecast cards
- 📅 5-day forecast
- 🎨 Dynamic background based on weather
- 🌙 Dark / Light mode
- 🌡️ Celsius / Fahrenheit toggle
- 💾 Save last searched city
- ⭐ Save favorite cities (localStorage)
- 🌫️ Air quality indicator
- 🌅 Sunrise / sunset info
- 💬 Weather-based short comments
- ⏳ Loading spinner & improved error handling
- 📱 Fully responsive design
- 📦 PWA support (manifest + service worker)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- OpenWeather API
- LocalStorage
- Geolocation API
- Service Worker
- Web App Manifest
